About the Project

UCSD Alert Depot is a real-time dashboard and map of UCSD Police Alerts designed to keep students and community members informed about safety incidents happening around campus. Our mission is to enhance campus safety awareness by providing an intuitive, geospatial representation of security events.

The project began as a response to the traditional text-based alerts that lack visual context, making it difficult for users to understand the proximity and patterns of campus incidents. By leveraging modern web technologies and artificial intelligence, we've created an interactive platform that transforms complex data into accessible visual information.

Key Features

Real-Time Alerts

Automatically scrapes and processes UCSD Police alerts as they're published, providing up-to-date information on campus safety incidents.

Interactive Mapping

Visualize incidents on an interactive map with filtering options for alert types, crime categories, and date ranges.

AI-Powered Geocoding

Utilizes local LLMs to automatically geocode locations from textual descriptions, even recognizing campus-specific landmarks and buildings.

Data Analysis

Identifies patterns, high-risk areas, and temporal trends to help users make informed decisions about campus safety.

Community Focus

Designed with UCSD students, faculty, and visitors in mind, with plans to expand to the broader San Diego area.

Data Export

Download incident data as CSV for further analysis or integration with other safety systems.

Interactive Demo

Experience our interactive map visualization of campus safety alerts. Filter by date, alert type, and crime category to see patterns and hotspots across the UCSD campus.

Technology Stack

Frontend

  • HTML5, CSS3, JavaScript
  • MapLibre GL JS for interactive mapping
  • Responsive design for all devices

Backend

  • FastAPI for high-performance API endpoints
  • Python for data processing and analysis
  • AsyncIO for real-time data handling

Data Processing

  • Beautiful Soup for web scraping
  • spaCy for natural language processing
  • Local LLMs (Mistral) for geocoding

Data Storage

  • JSON for lightweight data storage
  • CSV exports for data sharing
  • Geocoded location database
